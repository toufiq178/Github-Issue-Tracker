1️⃣ What is the difference between var, let, and const?

answer  :  In js there is 3 ways to declare variables : var, let, const

var  : if you use var , you can re-declare , update the variable.  
let  : if you use let , you can only update the variable.  
const  : if you use const , you can't update or re-declare variable.  


2️⃣ What is the spread operator (...)?

answer : spread operator expands an array or object into individual element of them and create shallow copy .


3️⃣ What is the difference between map(), filter(), and forEach()?

answer :  map(), filter(), and forEach() is array method .

map :  take an array then work on each element but doesn't return new array.

filter  :   take an array and apply a condition . if the condition is fulfilled, it creates a new array consisting of the items that meet that condition.
